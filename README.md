Фановый шаблон с разблокированным строительством и наймом из города юнитов до Т4 для тестирования финалок. Строим ветки, нанимаем отряды в Т5 городе и выходим финалиться! Без воров и жезловиков, 1 юнит - одна банка на 15%.

Содержимое архива скопировать в директорию игры\Templates.
Для корректной работы требуется версия SMNS 2.4 и выше
